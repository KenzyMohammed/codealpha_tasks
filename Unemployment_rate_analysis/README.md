# India Unemployment Data Analysis (2019–2020)

This project performs Exploratory Data Analysis (EDA) on Indian unemployment data before and during the COVID-19 period.

## Objective
To investigate how unemployment changed during the COVID-19 lockdown and measure the difference before vs during the pandemic.

## Key Insights
- Unemployment was stable before COVID (~9%)
- During COVID lockdown (Mar–Jun 2020) unemployment increased to ~17%
- Huge spike in April/May 2020 (national lockdown)
- Urban unemployment rate > Rural unemployment rate
- Impact was nationwide — all states showed the same shock

## Steps Performed
- Data cleaning & merging
- Handling missing data
- Visualizations (trend, seasonal, regional, rural vs urban)
- COVID impact comparison

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Conclusion
COVID-19 caused a major labour market shock in India, doubling the unemployment rate and affecting urban areas the most.
